what happens next

Tap save my gift below. A file called data.js downloads to your device — that's it, that's the whole gift, packed into one file.

Send that file to whoever set up the website (WhatsApp, email, however's easiest).

They open the site's GitHub page, click Add file → Upload files, drag data.js in, and click Commit changes. No coding.

The site updates on its own within about a minute. That's the whole process — you can come back and re-do this anytime.
